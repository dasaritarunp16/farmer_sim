# Farmer Sim

A narrative-driven 3D farming simulation built with Three.js. You play as **Richard Evans**, a farmer who lost everything to a devastating drought. You have **5 days** to harvest **120 bushels of wheat** — or lose the family farm forever.

---

## Overview

Farmer Sim is an interactive 3D web-based simulation where players manage resources, explore a farming village, and make decisions that shape the story across five in-game days. The game combines resource management with narrative storytelling in a fully rendered 3D environment.

## Features

- **3D Open World** — Explore a fully rendered farming village with a wheatfield, market, well, and home
- **Day/Night Cycle** — Dynamic sky and lighting that transitions across a 5-day timeline
- **Resource Management** — Balance wheat, gold, and water to meet your harvest goal
- **Narrative Dialogue System** — Location-based story events and character reflections that change each day
- **Journal** — A paper-style in-game journal that tracks your stats, story progress, and dialogue history
- **Player Choices** — Make decisions that affect crop yields and story outcomes
- **Win/Loss Condition** — Collect 120+ bushels to save the farm, or fall short and face foreclosure

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/dasaritarunp16/farmer_sim.git
cd farmer_sim

# Install dependencies
npm install

# Start the development server
npm run dev
